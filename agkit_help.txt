Usage: ag-kit init [options]

Install .agent folder into your project

Options:
  -f, --force          Overwrite if folder already exists (default: false)
  -p, --path <dir>     Path to the project directory (default:
                       "D:\\workcode\\studiowebsite")
  -b, --branch <name>  Select repository branch
  -q, --quiet          Suppress output (for CI/CD) (default: false)
  --dry-run            Show what would be done without executing (default:
                       false)
  -h, --help           display help for command
